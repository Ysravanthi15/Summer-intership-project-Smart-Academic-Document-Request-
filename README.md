# 🚀 Smart Academic Document Request System

## 📌 Project Overview

The **Smart Academic Document Request System** is designed to simplify and digitize the traditional academic document request process.

Instead of visiting the institution multiple times, students can simply **scan a QR code** and submit their request online, making the process faster and more convenient.

---

## ⚙️ Technologies Used

* QR Code Integration (linked to SharePoint Form)
* SharePoint (for data collection and storage)
* Power Automate (for workflow and notifications)
* Power BI (for dashboard and insights)
* Excel for data cleaning

---

## 🔄 Workflow

1. Students scan the QR code
2. Redirected to SharePoint Form
3. Submit document request
4. Data stored in SharePoint
5. Automated approval workflow triggered
6. Email notifications sent
7. Dashboard updates in Power BI
8. Document ready for collection

---
